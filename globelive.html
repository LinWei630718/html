📖 系統使用說明

🎬 導播端操作

1. 選擇畫質（標準/高清/超清）
2. 點擊「啟動攝影機」開始直播
3. 點擊導播 ID 即可自動複製
4. 將 ID 分享給觀眾
5. 當觀眾請求連線時會跳出確認視窗
6. 可選擇「開始錄影」保存直播內容
7. 即時查看線上人數和評分統計
8. 在觀眾名單中管理連線觀眾
9. 點擊全螢幕按鈕進入沉浸模式

👀 觀眾端操作

1. 輸入導播提供的 ID
2. 點擊「請求連線」等待導播同意
3. 連線成功後即可觀看直播
4. 查看線上觀眾人數和連線品質
5. 隨時給主播評分回饋
6. 使用全螢幕按鈕獲得最佳觀看體驗

📅 直播預告功能

導播可以設定倒數計時：
• 輸入時、分、秒設定開播時間
• 點擊「設定倒數計時」啟動
• 倒數期間觀眾會看到預告畫面
• 倒數結束後自動開始直播
• 可隨時點擊「取消倒數」提前開播

📹 畫質說明

• 標準 480p - 適合網路較慢時使用
• 高清 720p - 推薦使用，平衡畫質與流暢度
• 超清 1080p - 最佳畫質，需要良好網路環境

📶 連線品質指示

系統會自動偵測連線品質：
• ● 優異 - 延遲 < 100ms
• ● 良好 - 延遲 100-200ms
• ● 普通 - 延遲 200-500ms
• ● 不佳 - 延遲 > 500ms

💡 實用技巧

• 點擊導播 ID 輸入框即可快速複製
• 使用快速鍵 F11 也可進入全螢幕
• 建議使用耳機避免回音
• 確保良好的光線提升畫質
• 導播可在觀眾名單中踢除不當觀眾

🔧 技術需求

• 使用 Chrome/Edge/Safari 瀏覽器
• 允許攝影機與麥克風權限
• 建議使用 Wi-Fi 或 4G/5G 網路
• 支援 WebRTC 的現代瀏覽器
