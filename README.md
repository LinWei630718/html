# 暗棋 (darkchess.html)

這是一個基於 HTML5 開發的象棋遊戲，暗棋使用 4行×8列 共 32格 的棋盤，所有 32 顆棋子正面朝下隨機排列。玩家輪流翻棋、移動或吃子，消滅對方所有棋子者獲勝。

- **決定陣營**：第一個翻開棋子的顏色即為翻棋者的陣營。首翻後，翻棋者繼續走棋（首翻算行動），對方接著輪到。
- **棋子階級與吃子規則**：
  - **帥/將**：階級 7，最高，最強，可吃除兵/卒外所有棋子。
  - **仕/士**：階級 6，可吃象/相及以下。
  - **相/象**：階級 5，可吃車/俥及以下。
  - **俥/車**：階級 4，可吃馬/傌及以下。
  - **傌/馬**：階級 3，可吃炮/包及以下。
  - **炮/包**：階級 2，特殊移動：直線不越子，吃子：必須隔一砲台。
  - **兵/卒**：階級 1・反制，最弱，但可吃帥/將。
- **特殊規則**：
  - **🎯炮（包）的移動與吃子**：炮沿直線移動到空格（中間不能有任何棋子）。吃子時，炮與目標之間必須恰好隔一顆棋子（砲台），才可吃掉目標（不受階級限制）。
  - **⚡兵/卒 vs 帥/將**：兵（卒）可吃帥（將）；帥（將）無法吃兵（卒）。
  - **🃏 翻開暗棋**：翻棋視為一次行動，翻開後輪到對方。
  - **🏆 勝利條件**：消滅對方所有棋子（含未翻）即獲勝；若一方無棋可走，則該方認負。
- **操作說明**：
  - **翻牌**：點擊背面朝上的棋子即翻開。
  - **移動**：先點選己方棋子，再點選目標（綠點=可移動，紅框=可吃子）。
  - **悔棋**：每局最多 3 次，人機模式自動撤回 AI 挪步，
  - **快捷鍵**：R 重開・U / Ctrl+Z 悔棋・Esc 關閉視窗。



# 獵魔倖存者 (vampire.html)

這是一個基於 HTML5 開發的生存遊戲，靈感來自於 *Vampire Survivors*。玩家需要在不斷湧來的怪物潮中生存，收集經驗寶石升級，並取得強力的武器與被動技能。

## 遊戲特色

- **核心玩法**：自動攻擊、手動走位閃避。
- **升級系統**：
  - **武器**：皮鞭、魔杖、飛斧、聖經、火球等。
  - **被動道具**：菠菜 (增加傷害)、空虛之心 (增加血量上限)、魔法書 (減少冷卻) 等。
- **敵人多樣性**：蝙蝠、骷髏、幽靈、殭屍，以及在特定時間出現的 Boss 與死神。
- **操作方式**：
  - **鍵盤**：WASD 或 方向鍵移動。
  - **觸控**：支援虛擬搖桿 (適合手機/平板)。
- **遊戲目標**：存活 10 分鐘以獲得勝利。
