# 獵魔倖存者 (Demon Hunter Survivors)

這是一個基於 HTML5 開發的生存遊戲，靈感來自於 *Vampire Survivors*。玩家需要在不斷湧來的怪物潮中生存，收集經驗寶石升級，並取得強力的武器與被動技能。

## 遊戲特色

- **核心玩法**：自動攻擊、手動走位閃避。
- **升級系統**：
  - **武器**：皮鞭、魔杖、飛斧、聖經、火球等。
  - **被動道具**：菠菜 (增加傷害)、空虛之心 (增加血量上限)、魔法書 (減少冷卻) 等。
- **敵人多樣性**：蝙蝠、骷髏、幽靈、殭屍，以及在特定時間出現的 Boss 與死神。
- **操作方式**：
  - **鍵盤**：WASD 或 方向鍵移動。
  - **觸控**：支援虛擬搖桿 (適合手機/平板)。
- **遊戲目標**：存活 10 分鐘以獲得勝利。
